# PourSort

**Ad-free color sorting puzzle game for iOS.**

## What It Is

PourSort is a clean, ad-free ball/color sorting puzzle game. Players sort colored balls into tubes until each tube contains only one color. Algorithmically generated levels provide endless content — 1000+ levels from day one.

## Current State

Research phase — M001 in progress.

## Stack

- SwiftUI (no SpriteKit needed — pure SwiftUI animations)
- SwiftData (level progress, user data)
- StoreKit 2 (native IAP)
- iOS 17+
- No third-party dependencies

## Core Principles

- **No ads ever** — premium model only
- **Algorithmic level generation** — seed-based, deterministic, solvable
- **1000+ levels at launch** — no content bottleneck
- **Satisfying animations** — pour/flow effects, haptics, celebrations

## Genre

Casual puzzle — color sort / ball sort / tube sort
